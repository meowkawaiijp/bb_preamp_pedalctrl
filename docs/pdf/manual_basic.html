各部の名称と役割

ディスプレイ (LCD)

現在のプリセット番号、各パラメータ（Volume, Gain, Treble, Bass）の値、および現在の操作状態を表示します。

エンコーダー

  • PARAM エンコーダー (右): 選択されているパラメータの値を調整します。押し込むと決定(ENTER)ボタンとして機能します。
  • PRESET エンコーダー (左): プリセット番号(0-9)を素早く切り替えます。回すと即座にプリセットが読み込まれます。

コントロールボタン

  • LEFT / RIGHT ボタン: 調整したいパラメータ（項目）を選択します（カーソル移動）。
  • ENTER ボタン (PARAMエンコーダー押し込み):
    • 短押し：プリセットの保存確認など、決定操作を行います。
    • 長押し：設定モード選択画面（Change Mode）へ移行します。

フットスイッチ

設定により動作が異なります（システム設定参照）。

  • Preset Mode: 押すたびに次のプリセットへ切り替わります。
  • Alternate Mode: エフェクトのON/OFF（Bypass）を切り替えます。

プレイモード (Play Mode)

電源投入時の標準的な状態です。このモードで演奏とリアルタイムの音色調整を行います。

調整可能なパラメータ

パラメータ 範囲 説明
Volume 0 〜 99 出力音量を調整します。
Gain 0 〜 99 歪みの強さを調整します。
Treble -50 〜 +50 高域のブースト/カットを行います。
Bass -50 〜 +50 低域のブースト/カットを行います。
ヒント: パラメータを変更すると、ディスプレイ上の表示が「変更あり」の状態になります。保存しない限り、プリセットを切り替えると変更内容は破棄されます。

Hardware Interface & Play Mode

はじめに 基本操作 プリセット管理 WiFi・Webエディタ システム設定・MIDI

各部の名称と役割

ディスプレイ (LCD)

現在のプリセット番号、各パラメータ（Volume, Gain, Treble, Bass）の値、および現在の操作状態を表示します。

エンコーダー

  • PARAM エンコーダー: 選択されているパラメータの値を調整します。
  • PRESET エンコーダー: プリセット番号(0-9)を素早く切り替えます。

コントロールボタン

  • LEFT / RIGHT ボタン: 調整するパラメータを選択（カーソル移動）します。
  • ENTER ボタン:
    • 短押し：プリセットの保存確認画面へ進みます（変更がある場合）。
    • 長押し：設定モード選択画面（Change Mode）へ移行します。

フットスイッチ

設定により「プリセットの順次切り替え」または「エフェクトのON/OFF」として動作します。

プレイモード (Play Mode)

電源投入時の標準的な状態です。このモードで演奏とリアルタイムの音色調整を行います。

パラメータ 範囲 説明
Volume 0 〜 99 出力音量を調整します。
Gain 0 〜 99 歪みの強さを調整します。
Treble -50 〜 +50 高域のブースト/カットを行います。
Bass -50 〜 +50 低域のブースト/カットを行います。
ヒント: パラメータを変更すると、ディスプレイ上の表示が「変更あり」の状態になります。保存しない限り、プリセットを切り替えると変更内容は破棄されます。

WiFi・Webエディタ

Remote Editing via Browser

はじめに 基本操作 プリセット管理 WiFi・Webエディタ システム設定・MIDI

WiFiの準備

本機はWiFi経由でブラウザから設定を行うことができます。

1. WiFiを有効にする

設定モード（Settings）で「WiFi: ON」になっていることを確認してください。変更した場合は再起動が必要です。

2. SSIDとパスワードの設定

  1. ENTER ボタンを長押しして「Change Mode」に入ります。
  2. 「Edit SSID」または「Edit PASS」を選択して ENTER ボタンを押します。
  3. エンコーダーと方向ボタンを使って、接続するWiFiルーターの情報を入力します。
  4. 入力後、保存して本体を再起動します。

Webエディタへの接続

  1. 本機と同じWiFiネットワークにスマートフォンやPCを接続します。
  2. ブラウザを開き、アドレスバーに http://bbp.local/ と入力します。
  3. 「BBP Preset Editor」画面が表示されます。

Webエディタの使い方

ブラウザ上のスライダーを動かすことで、本体のプリセット値をリアルタイムに変更できます。

  • スライダー操作: 各プリセットのパラメータを調整します。
  • Save All Presets: 全てのプリセット設定を一括で本体に保存します。
注意: 演奏中にWiFi通信によるノイズが発生するのを防ぐため、プレイモード中の一部動作では通信を抑制する仕様になっています。

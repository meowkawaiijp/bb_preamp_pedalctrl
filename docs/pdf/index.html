はじめに

この度は、Presettable BB-Preamp Controllerをお買い上げいただき、誠にありがとうございます。

本機は、名機Blues Breakerプリアンプのサウンドをベースに、10個のプリセット保存機能、Webブラウザからのリモート編集、MIDIコントロールなどの現代的な機能を搭載したエフェクター・コントローラーです。

主な特徴

  • 10プリセット保存: お気に入りの設定を最大10個まで保存し、瞬時に呼び出すことができます。
  • 直感的な操作: 本体に搭載された液晶ディスプレイとエンコーダーにより、直感的なパラメータ調整が可能です。
  • Webエディタ: WiFi経由でスマートフォンやPCからプリセットの編集・保存が可能です。
  • MIDI対応: MIDIプログラムチェンジメッセージにより、外部機器からプリセットを切り替えられます。
  • 柔軟なフットスイッチ: プリセットの順次切り替え、またはエフェクトのON/OFF切り替えを選択可能です。

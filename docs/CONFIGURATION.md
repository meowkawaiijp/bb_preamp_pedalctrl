# Configuration and ranges

## Preset parameters
- volume: 0..99
- gain: 0..99
- treble: -50..50
- bass: -50..50

## App parameters
- presetNo: 0..(NUM_PRESET - 1)
- midi_ch: 1..16
- footsw_mode: 0 = preset, 1 = alternate
- wifi_enable: 0/1
- preset_max: 2..(NUM_PRESET - 1)

## WiFi credentials
- SSID length: up to 32 characters
- PASS length: up to 63 characters

# Presettable BB-Preamp Controller
 BB-Preamp pedal controller programmed using Arduino<br>
 View more info at http://toshi.life.coocan.jp/review/en_diy_arduino_bbpreamp.html<br>

## Overview
- Arduino-based controller for a BB-Preamp pedal.
- Target MCU: ESP32C3 (Seeed XIAO ESP32C3 in the source comments).
- Supports local UI, MIDI program change, and a WiFi web UI.
